# Sistema CRUD em PHP

Projeto pessoal desenvolvido com foco em back-end, utilizando PHP e MySQL.

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Edição de registros
- Exclusão de usuários
- Prepared Statements

## Tecnologias
- PHP
- MySQL
- HTML5
- CSS3

## Objetivo
Projeto para estudo e prática de desenvolvimento back-end.
